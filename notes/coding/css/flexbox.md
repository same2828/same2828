# Flexbox

# flex 
```css
flex: none | [ <'flex-grow'> <'flex-shrink'>? || <'flex-basis'> ];
```

- Shorthand for 
```css
flex: flex-grow, flex-shrink, flex-basis;
```
- Default
```css
flex: 0, 1, auto;
```

# flex-wrap
```css
flex-wrap: wrap;
```

# justify-content
```css
justify-content: space-between;
```
